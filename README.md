# libjsjson
A C++14 JSON Parser and Serializer

To build:

Install cmake 3.9.4 or above: https://cmake.org

```
cd libjsjson
mkdir build
cd build
cmake ..
make
```
Creates the following files in libjsjson/bin folder:

libjsjson.a
test_libjsjson

To test:

```
cd build
ctest
```

Or:

```
../bin/test_libjsjson
```
